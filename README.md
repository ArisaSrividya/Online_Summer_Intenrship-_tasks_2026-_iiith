Online Summer Internship - FFmpeg Tasks

This repository contains tasks completed as part of an internship involving video processing using FFmpeg.

WEEK-1

Task 1 - Frame Extraction
Extracted frames from a video using FFmpeg. Sample frames are available in the task-1 folder.

Task 2 - Frame Reconstruction
Generated approximately 1800 frames (30 fps for 1 minute) and reconstructed a video from those frames using FFmpeg.

Task 3 - Audio Merge
Merged a one-minute audio clip with the reconstructed video using FFmpeg.
